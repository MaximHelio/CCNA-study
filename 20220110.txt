- VLSM (Variable Length Subnet Mask, 가변 길이 서브넷 마스크)
: 개요
	; 서브넷 길이가 다른 여러 개의 서브넷에서 동일한 네트워크 번호로 다른 서브넷 마스크를 지정할 수 있는 특성을 말한다.
	; VLSM variable-length subnet mask(가변 길이 서브넷 마스크)의 약어.
	; 서로 다른 서브넷에서 동일한 네트워크 번호로 다른 서브넷 마스크를 지정할 수 있는 특성. 
	; VLSM은 가용 주소 공간을 최적화하는데 도움이 된다.
	; 기존의 서브넷이 동일한 주소의 길이로 구분하는 방법인 반면, 서브넷된 주소를 다시 서브넷팅한다. 


: 실습(Module 00. 46p)
	
	; step1.  현재 주어진 대역은 '192.168.20.0/24' 이다. 이 대역을 8개의 대역으로 서브네팅해라.
	 	
		192.168.20.0/27
		192.168.20.32/27
		192.168.20.64/27
		192.168.20.96/27
		192.168.20.128/27
		192.168.20.160/27
		192.168.20.192/27
		192.168.20.224/27
	
	  step2. 위의 대역 중 7번째 대역(192.168.20.192/27)을 8개 대역으로 다시 한 번 서브네팅하라.

		27 + 3bit (8개 대역의 VLSM) = 30bit
		32bit(IPv4 전체 bit 수) - 30 = 2bit(호스트에 부여할 IP)
		각 대역별 SM 	: 255.255.255.252 (128+64+32+16+8+4)
		각 대역별 CIDR	: 192.168.20.192 /30 (8+8+8+6)

- CIDR(Classless Inter-network Domain Routing)
: 개요
	; CIDR (Classless Inter-network Domain Routing)이란 주소 재할당 개념이다. 
	기존 Class기반 주소에서 Class를 제외(실제 Class를 적용하지 않는다는 것이 아님)하고 
	32bit 전체 bit에 대해 Network과 Host를 재설정한 주소 구조이다. 
	기존 Class 기반 주소에 비해 주소 손실을 줄여주고,
	Router에는 구조화된 주소 할당으로 인해 Routing Table을 줄여 packet Delay를 줄인다.
	
	; 정적(Static)라우팅보다 동적(Dynamic)라우팅에 적용된다.
	; 표기법은, '/[서브넷 개수의 비트수의 합]'로 표기한다.
	

- Cisco Device와 IOS
: Cisco 장비 종류
	; Switch
		-> 개요
			: OSI 7 Layer의 2계층에 속하고, LAN을 구축하는 기본 장비
			: 중앙에 위치하고, End-Device(PC)를 연결시키는 역할
			: Multi Layer Switch (3계층 이상의 역할을 할 수 있는 스위치)
		
		-> 기본 기능
			: MAC Address의 학습
			: 루프 감지
			: VLAN(Virtual LAN)
		-> 역할
			: MAC Address를 구분할 수 있고, 이를 확인해서 해당하는 포트로만 
			Frame(데이터)을 전달

		-> Catalyst Switch Series
			: Catalyst(제품명)
			: 앞 2자리(성능을 의미, 숫자 클수록 고성능 기능)
			: 앞 2자리 이외(확장성을 의미)

	; Router
		-> 개요
			: 한 개의 네트워크는 반드시 라우터가 1개만 존재한다.
			: 네트워크(WAN)와 네트워크(WAN)를 연결시켜주는 장비(서로 다른 대역)
			: OSI 7 Layer의 3계층에 속한다.
			: 최초 출발지에서 최종 목적지까지 가는 최적의 경로를 설정하는 장비이다.
			: 내부망(LAN)의 PC에서 넘어오는 Frame을 외부망(WAN)과 연결시켜준다.
